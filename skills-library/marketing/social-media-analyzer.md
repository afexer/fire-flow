# Social Media Analyzer

## Triggers
Use when: social media performance, engagement rate, campaign analytics, Instagram metrics, LinkedIn analytics, TikTok performance, social ROI

---

## Core Metrics

**Engagement Rate formula:**
```
ER = (Likes + Comments + Shares + Saves) / Reach × 100
```

**Platform benchmarks:**

| Platform | Average ER | Good | Excellent |
|----------|-----------|------|-----------|
| Instagram | 1.22% | 3–6% | >6% |
| Facebook | 0.07% | 0.5–1% | >1% |
| Twitter/X | 0.05% | 0.1–0.5% | >0.5% |
| LinkedIn | 2.0% | 3–5% | >5% |
| TikTok | 5.96% | 8–15% | >15% |

---

## ROI Metrics

```
CPE (Cost Per Engagement) = Total Spend / Total Engagements
CPC (Cost Per Click)      = Total Spend / Total Clicks
ROI                       = (Value Generated - Spend) / Spend × 100
```

**Engagement value estimates (baseline):**
| Action | Estimated Value |
|--------|----------------|
| Like | $0.50 |
| Comment | $2.00 |
| Share | $5.00 |
| Save | $3.00 |
| Click | $1.50 |

---

## Analysis Workflow

1. **Collect data:** Pull last 30/60/90 days from platform analytics
2. **Calculate ER:** Per post and average for the period
3. **Identify top performers:** Top 20% of posts by ER
4. **Pattern analysis:** What do top posts have in common? (format, topic, length, time)
5. **Benchmark:** Compare against platform averages above
6. **Recommendations:** Double down on what works, cut what underperforms

---

## Content Performance Scoring

Score each post across:
- **Reach score:** Impressions vs. follower count ratio
- **Engagement score:** ER vs. platform benchmark
- **Conversion score:** Clicks or link taps (if trackable)
- **Save rate:** Saves / Reach (high saves = high-value content)

**Winning post signals:**
- ER 2× platform average
- Save rate >2%
- Comments with questions (signals community interest)
- Shares (signals shareworthy insight)
